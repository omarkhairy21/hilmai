## Key Implementation Files

### Bot Core
- `agent/src/bot.ts` - Main bot entry point, command handlers, message routing
- `agent/src/mastra/index.ts` - Mastra configuration and agent registry

### Mode Management
- `agent/src/lib/user-mode.ts` - Mode state management and helpers
- `agent/src/lib/messages.ts` - All user-facing messages including mode descriptions

### Subscription
- `agent/src/services/user.service.ts` - User CRUD, subscription checks
- `agent/src/services/subscription.service.ts` - Stripe integration
- `agent/supabase/schema.sql` - Database schema with subscription fields

### Workflows & Agents
- `agent/src/mastra/workflows/message-processing-workflow.ts` - Main message workflow
- `agent/src/mastra/agents/` - Mode-specific agents (logger, query, chat)

### Supporting Libraries
- `agent/src/lib/prompt-cache.ts` - Response caching (LibSQL)
- `agent/src/lib/embeddings.ts` - Vector search and embeddings
- `agent/src/lib/currency.ts` - Multi-currency support
- `agent/src/services/progress.service.ts` - Progress message updates
